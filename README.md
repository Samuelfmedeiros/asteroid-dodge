Desvie de asteroides com a nave! Jogo de reflexos no espaço.

## Como Jogar
- Movimente a nave com o mouse (ou toque no mobile)
- Desvie dos asteroides ☄️
- Cada asteroide desviado = +1 ponto
- A cada 30 pontos, a velocidade aumenta
- 3 vidas — Game Over quando acabar

## Stack
- React 18 (CDN) · CSS puro com @keyframes · requestAnimationFrame

## Executar
```bash
git clone git@github.com:Samuelfmedeiros/asteroid-dodge.git
cd asteroid-dodge
open index.html
```
